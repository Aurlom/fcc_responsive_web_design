Introduction

C is a procedural language develloped by Dennis Ritchie in 1972 at AT&T Bell Laboratories. It was initially developed as a system programming language to write the UNIX operating system. It is widely used today to program operating systems and compilers.

C is useful for these tasks due to it's "low-level" nature while remaining portable and not being as cryptic as assembly language.

Advantages include:

  • General purpose and widely compatible
  • Low-level and intentional memory management
    • This can also be a drawback as it's easy to leave vulnerabilities if not careful
  • Fast
  • Code organized into bracketed blocks

Why learn C

Learning C, especially as a first language, can teach a programmer good habits and help engender understanding of the underlying function of software. Having to specifically allocate memory, understand pointers, and have to construct strings as an array of characters teaches how data is stored and accessed, and how memory usage is directly related to the design of a program.

In addition, many modern languages use C syntax and concepts, including Java, PHP, JavaScript, C++, C#, and many others. C++ is almost a superset of C, and most C programs could compile in a C++ compiler.

C and C++

C++ was created to bring concepts of "object-oriented programming" into C. They share much of the same syntax, but C++ support of classes and features such as exception handling makes it more convenient for programming applications. That convenience comes at the price of speed, and thus, C remains popular for lower level programming.

Hello world

To get started with C, open a text editor and write the traditional "Hello, world!" code: #include <stdio.h> int main(void) { printf("Hello, world!"); return 0; }

  • #include <stdio.h>
    • A "preprocessor directive" telling the C compiler to include the functions from the stdio header file (which includes printf())
  • int main(void)
    • The main function is what executes first when you run a C program
    • "int" is a return type (integer) which allows the program to return error codes
    • "main()" is the name of the function, with main being a special reserved function for the main execution block
    • (void) are the arguments main takes in at execution, with void meaning nothing
  • { ... }
    • The braces under the main function bracket the code block belonging to the function
    • White space (except where absolutely necessary) is ignored by C compilers, and the braces tell C what code belongs to which functions
    • The above program could be written identically as follows: #include<stdio.h>int main(void){printf("Hello, world!");return 0;}
    • Judicious use of white-space makes programs more human readable
  • printf("Hello, world!");
    • The heart of our program. This line calls a function "printf()" and gives it a single argument, the literal string "Hello, world!"
    • Printf() is defined in the stdio library and its job is to print a "formatted string" to the console window
    • The statement is terminated by use of a semicolon, which tells the C compiler that the statement is complete
  • return 0;
    • The main function's "return value"
    • Because main() is the main body of the program, it isn't strictly necessary to return any value
    • Typically, a return value allows you to call a function and get something back
    • The return value for the main function only goes to whatever process started the program in the first place. This becomes useful when you want your program to return an error code when something goes wrong
    • By convention, returning a code of "0" means everything executed correctly.

Variables

Variables are storage containers for values in C and are declared as follows: type name = value;

For example: int x = 10;

Type indicates what kind of data the variable will store and can be one of the following basic types:

  • int - Stores an integer
  • char - Stores a single ASCII character
  • float - Stores a "floating point" number (a number with a decimal point)
  • double - Similar to a float, but uses more memory and thus allows for double precision
  • void - A type to indicate nothing

Name is how the variable is referenced later in the program, and is subject to the following rules:

  • Contains only letters, digits, and underscores
  • Begins with a letter or an underscore
  • Case-sensitive
  • Cannot contain white-space or special characters other than underscore
  • Reserved keywords (ex. "int") cannot be used

In addition, it is good practice to make variable names descriptive of their purpose, use all lower case letters (except for the special case of a constant), and to use "snake case" for multiple words ("words seperated by underscores"). For example: my_variable

Conditionals

Conditionals allow a program to execute on exclusive branching paths based on a defined condition using the "if else" blocks. if (condition 1) { path 1; } else if (condition 2) { path 2; } else { path 3; }

The "else" block executes when no other conditions are met.

Conditions take the form of boolean expressions that evaluate to "true" or "false" and use the following operators:

  • == - Equal to
  • != - Not equal to
  • < - Less than
  • > - Greater than
  • <= - Less than or equal to
  • >= - Greater than or equal to

Additional logical operators allow the combination or negation of conditional expressions:

  • && - Logical and
  • || - Logical or
  • ! - Negation
